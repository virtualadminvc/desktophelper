# 桌面提醒 \+ 计时器

一个简洁的桌面工具，用于显示电脑运行时间和自定义任务清单，支持外观与文本样式自定义。

## ✨ 功能特性

- **显示电脑运行时长**：在桌面以秒为单位展示系统总运行时间。

- **自定义提醒内容**：编辑并同步日常任务/待办事项到桌面。

- **灵活的外观设置**：
        

    - 调节窗口背景颜色与透明度。

    - 自定义计时器文字颜色与字号。

    - 自定义提醒文本颜色与字号。

- **最小化到托盘**：将窗口最小化至系统托盘，不占用桌面空间。

- **开机自启**：支持通过托盘菜单设置开机自动启动。

## 🚀 使用方法

1. **编辑提醒内容**：
        在“桌面提醒内容”文本框中输入你的任务或待办事项。

2. **调整外观样式**：
        通过“窗口外观”设置背景颜色和透明度。通过“计时器样式”调整计时器文字的颜色和大小。通过“提醒文本样式”调整提醒文字的颜色和大小。

3. **同步到桌面**：
        点击“立即同步到桌面”按钮，更新修改内容（无弹窗提示）。

4. **窗口管理**：
        点击“隐藏桌面窗口”或“显示桌面窗口”按钮切换桌面显示状态。点击右上角“—”按钮将窗口最小化到系统托盘。右键托盘图标可进行操作：显示/隐藏窗口、开机自启、退出程序。

## 📝 注意事项

- 编辑提醒内容后，需点击**立即同步到桌面**才能生效。

- 桌面窗口可拖拽边缘调整大小，内容自动换行，无滚动条。

- 背景颜色和透明度统一控制整个窗口。

- 计时器和提醒文本的字号可分别调节，调节过程流畅不卡顿。

- 右上角“—”按钮最小化到托盘，右键托盘图标可执行更多操作。
